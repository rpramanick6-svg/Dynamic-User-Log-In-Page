# Dynamic User Log in Page
I built this project as a simple and interactive Login / Register interface using HTML, CSS, and JavaScript. The main goal of this project was to create a smooth UI that allows users to switch between the Sign In and Sign Up sections with an animated transition.

## Overview

In this project, I worked on connecting JavaScript functionality with a styled frontend interface. The UI uses a panel-based layout where the visible section changes based on user interaction, while CSS handles the design, positioning, and transitions.

## Features

- I created a Sign In and Sign Up interface in a single layout.
- I implemented a toggle-based transition between both panels.
- I used JavaScript to control the UI state dynamically.
- I used CSS animations and styling to give the interface a modern look.

## My Contribution

I primarily focused on developing the JavaScript functionality in this project. I wrote the logic that listens for button clicks and changes the state of the main container by adding or removing the `active` class.

For the CSS styling and overall design, I took help from a YouTube tutorial and AI assistance. My main learning focus was on understanding how JavaScript functions can be used to control frontend behavior and work together with CSS-based transitions.

## Technologies Used

- HTML
- CSS
- JavaScript

## JavaScript Logic

In this project, I used JavaScript to select the main container and the Register/Login buttons. When the Register button is clicked, I add the `active` class to the container. When the Login button is clicked, I remove the `active` class.

This class-based approach helped me understand how JavaScript can control UI changes without rewriting the whole layout.
